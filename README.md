### installation
1. install gnu stow for your distro
2. clone the repo
   ```sh
   git clone https://github.com/drem-jpg/dots.git
   ```
3. cd into repo
   ```sh
   cd dots
   ```
4. use stow to create symlinks
   ```sh
   stow .
   ```
